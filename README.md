Запустить локально: 

- склонировать репу в локальную машину 
- выполнить данные скрипты 
- poetry shell 
- poetry install
- cd /Tests/ui_tests
- pytest --browser=local_chrome

- склонировать репу в локальную машину 
- выполнить данные скрипты
